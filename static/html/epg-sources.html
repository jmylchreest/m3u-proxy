EPG (Electronic Program Guide) Sources allow you to configure where to obtain program schedule data for your channels. This data is used by the proxy generator to create XMLTV format EPG files that can be consumed by media players and PVR applications.

How it works:

  • Source Configuration: Define EPG data sources (URLs, APIs, files) that provide program information
  • Channel Matching: Map EPG channels to your filtered Stream Sources using various matching criteria
  • XMLTV Generation: The proxy generator automatically creates XMLTV format EPG files using the Rust xmltv crate
  • Integration: Generated EPG data is served alongside your M3U playlists for seamless integration

Supported EPG Source Types:

  • XMLTV files (local or remote URLs)
  • JSON APIs with program data
  • Third-party EPG services
  • Custom scrapers and importers

Technical Implementation: Uses the xmltv Rust crate for efficient parsing and generation of XMLTV data, with support for large files through streaming processing to minimize memory usage.
